## 🛠️ 环境要求

### 推荐环境
- **操作系统**: Linux (Ubuntu/CentOS) 或 macOS
- **Shell**: Bash 4.0+ 或 Zsh
- **编辑器**: VS Code、Vim 或 Nano
- **终端**: 支持颜色显示的现代终端

### Windows用户
- 使用WSL (Windows Subsystem for Linux)
- 或使用Git Bash
- 或使用Docker容器

### 必需工具
```bash
# 检查Bash版本
bash --version

sudo apt-get install curl wget jq bc
```

## 📖 使用说明

### 学习建议
1. **循序渐进**: 按照学习路径顺序学习，不要跳跃
2. **动手实践**: 每个示例都要亲自运行和修改
3. **完成练习**: 认真完成每个阶段的练习题目
4. **项目实战**: 至少完成一个完整的实战项目
5. **持续练习**: 在实际工作中应用所学知识

### 代码运行
```bash
# 给脚本添加执行权限
chmod +x script.sh

# 运行脚本
./script.sh

# 或者直接用bash运行
bash script.sh
```

### 调试技巧
```bash
# 启用调试模式
bash -x script.sh

# 在脚本中启用调试
set -x  # 开启调试
set +x  # 关闭调试

# 检查语法
bash -n script.sh
```
